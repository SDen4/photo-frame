# Photo Frame

## Start this project:
```sh
$ git clone https://github.com/SDen4/photo-frame.git
$ cd photo-frame.git
$ npm i
$ git init
$ git remote set-url origin <link of new repo in github>
$ git push -u origin master
$ npm run dev
```
